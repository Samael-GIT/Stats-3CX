# Stats-3CX

Stats-3CX est un outil développé en Python permettant de collecter, analyser et visualiser des statistiques liées à l'utilisation de 3CX, une solution de communication professionnelle.

## Fonctionnalités

- **Collecte de données** : Extraction automatisée des données d'utilisation de 3CX.
- **Analyse** : Traitement et analyse des données pour obtenir des insights exploitables.
- **Visualisation** : Génération de graphiques et rapports pour une meilleure compréhension des performances.

## Prérequis

Avant d'utiliser cet outil, assurez-vous d'avoir installé les éléments suivants :

- **Python 3.x** (dernière version recommandée)
- **Bibliothèques Python nécessaires** (liste dans `requirements.txt`)

Pour installer les dépendances, exécutez la commande suivante :
```bash
pip install -r requirements.txt
```

## Installation

1. Clonez ce dépôt sur votre machine locale :
   ```bash
   git clone https://github.com/Samael-GIT/Stats-3CX.git
   ```
2. Accédez au répertoire du projet :
   ```bash
   cd Stats-3CX
   ```

## Utilisation

Lancez le script principal :
   ```bash
   python main.py
   ```
